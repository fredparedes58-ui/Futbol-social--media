Stack IA

AGENT 03

Match Predictor

AGENT 04

Genera formación + plan de partido contextualizado al rival, terreno y XI disponible esa jornada.

Preguntas

FAQ

¿Funciona offline?

Sí. GRADA es una app nativa con almacenamiento local en SQLite. Las pantallas principales funcionan sin red y la sincronización con el servidor se reanuda automáticamente al reconectar.

¿Qué stack usa por dentro?

React Native + TypeScript + Expo + Supabase (auth, Postgres, Realtime, Storage) + TanStack Query v5 + React Navigation + Reanimated + Push notifications nativas (APNs / FCM).

¿Cómo es la seguridad de los datos?

Row-Level Security en Supabase para todas las tablas, JWT por usuario, roles `coach`/`player`/`parent` con permisos granulares y backups diarios automáticos.

¿Qué es la liga FFCV?

Federación de Fútbol de la Comunidad Valenciana. La app trae cargada la temporada juvenil con 13 equipos reales y permite seguir clasificación, jornadas y goleadores en vivo.

¿Los agentes IA son deterministas?

Sí. Cada agente (Coach, Scouting, Predictor, Tactic) tiene un prompt versionado, temperature controlada y RAG sobre el corpus del equipo para reproducibilidad y trazabilidad.

¿Cómo se descarga?

GRADA es una app nativa. Se descarga directamente desde la App Store (iOS) y Google Play (Android). Icono propio, splash screen, push notificaciones, biometría y acceso a cámara como cualquier app nativa.

¿Hay app para padres?

Sí. El rol `parent` accede a feed, avisos urgentes, calendario de partidos, asistencia y notificaciones push. Sin permisos de edición.

¿Puedo migrar mis datos actuales?

Sí. Importador CSV para plantilla, calendario y stats históricas. Soporte dedicado en plan Club y Federación para la migración inicial.
